shadcn Dashboard Registry

Reusable dashboard components for shadcn/ui with neutral theme

Installation

Add the registry to your components.json:

{ "registries": { "@dashboard": "https://leobrival.github.io/shadcn-dashboard-registry/r/{name}.json" } }

Components

dashboard-sidebar
Collapsible sidebar with navigation, user menu, and theme support
npx shadcn@latest add @dashboard/dashboard-sidebar
dashboard-header
Responsive header with breadcrumbs, search, and user actions
npx shadcn@latest add @dashboard/dashboard-header
dashboard-layout
Complete dashboard layout combining sidebar, header, and content
npx shadcn@latest add @dashboard/dashboard-layout
stats-card
Statistics card with icon, value, and trend indicator
npx shadcn@latest add @dashboard/stats-card
data-table
Feature-rich data table with sorting, filtering, and pagination
npx shadcn@latest add @dashboard/data-table
empty-state
Placeholder for empty data states with icon and action button
npx shadcn@latest add @dashboard/empty-state
